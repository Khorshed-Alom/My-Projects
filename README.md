## 🧰 Tech Stack

- **Language:** Solidity `^0.8.x`
- **Framework:** [Foundry](https://book.getfoundry.sh/)
- **Testing:** Forge
- **Package Manager:** `forge install`
- **Chain Compatibility:** Ethereum, Polygon, Arbitrum, and other EVM-based chains
